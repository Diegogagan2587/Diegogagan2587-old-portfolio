CSS Flexbox: Usa la Propiedad flex-wrap para envolver una fila o columna

CSS flexbox tiene una caracteristica para dividir un articulo flex en multiples filas (o columnas). Por defecto un contenedor flex ajustara todos los articulos flex juntos. Por ejemplo, una fila estara toda en una linea.

Sin embargo, Usar la Propiedad flex-wrap le dice a CSS que envuelva articulos. Esto significa articulos extra moviendose dentro de una nueva fila o columna. El punto de quiebre de donde sucede la envoltura depende de el tamaño de el articulo y el tamaño de el contenedor.

CSS ademas tiene una opcion para la direccion de la envoltura:

  • nowrap: Esta es la configuracion por defecto, y no envuelve articulos.
  • wrap: Envuelve articulos de la izquierda a la derecha si ellos estan en una fila, o de arriba hacia abajo si ellos estan en una columna.
  • wrap-reverse: Envuelve articulos de la derecha a la izquierda si ellos estan en una fila, o del abajo hacia arriba si ellos estan en una columna.

El diseño actual tiene tantas cajas para una fila. Añade la Propiedad CSS flex-wrap a el elemento #box-container, y dale un valor de wrap
