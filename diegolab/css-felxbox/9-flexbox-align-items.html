CSS Flexbox: Alinea Elementos Usanto la Propiedad "align-items"

Â¿Que es "align-items"?

La Propiedad align-items es similar a justify-content. Recuerda que la propiedad justify-content alineo articulos flex a lo largo de el eje principal(main axis). Para las filas, el eje principal (main axis) es una linea horizontal, y para las columnas es una linea vertical.

Contenedores Flex ademas tienen un eje transversal (cross axis) el el cual es el opuesto al eje principal (main axis). Para las filas(rows) el eje transversal(cross axis) es vertical, y para las columnas el eje transversal es horizontal.

CSS ofrece la Propiedad align-items para alinear articulos a lo largo de el eje transversal cross axis. Para una fila (row), esta dice a CSS como empujar los articulos en toda la fila arriba o abajo dentro de el contenedor. Y para una columna, como empujar todos los items a la izquierda o derecha dentro de el contenedor.

Los diferentes valores disponibles para align-items incluyen

  • flex-start: Alinea articulos a el inicio de el contenedor flex. Para filas, esto alinea articulos a la cima de el contenedor. Para las columnas, esto alinea articulos a la izquierda de el contenedor.
  • flex-end: Alinea articulos a le fin de el contenedor flex. para las filas, esto alinea articulos a el fondo de el contenedor. Para las columnas, esto alinea articulos a la derecha de el contenedor.
  • center: Alinea articulos a el centro. para filas, esto alinea articulos vertical mente (espacio igual arriba y debajo de el articulo). Para columnas, Esto las alinea horizontalmente (igual espacio a la izquierda y derecha de los articulos).
  • stretch: Estira los articulos para rellenar el contenedor flex. Por ejemplo, filas de articulos son estiradas para rellenar el contenedor flex de arriba hacia abajo. Este es el valor predeterminado si no hay un valor especificado para align-items.
  • baseline: Alinea elementos a sus lineas base. La linea base es un concepto de texto, piensa de el como la linea que sostienen a las letras o donde las letras se sientan.

Un ejemplo ayudara a mostrar esta propiedad en accion. Agrega la propiedad CSS align-items a el elemento #box-container, y dale un valor de center.
