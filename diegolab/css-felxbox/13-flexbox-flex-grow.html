CSS Flexbox: Usa la propiedad "flex-grow" para expandir articulos

Lo opuesto a flex-shrink es la propiedad flex-grow. Recuerda que flex-shrink controla el tama単o de los articulos cuanto el contenedor se encoge. La propiedad flex-grow controla el tama単o de los articulos cuando el contenedor padre se expande.

Usando un ejemplo similar de el ultimo desafio, si un articulo tiene un valor flex-grow de 1 y el otro tiene un valor flex-grow de 3 , el que tiene el valor de 3 crecera tres veces como mucho a comparacion de el otro.

Agrega la propiedad CSS flex-grow tanto a #box-1 como a #box-2. Dale un valor de 1 a #box-1 y un valor de 2 a #box-2.
