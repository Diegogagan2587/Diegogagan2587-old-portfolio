CSS Flexbox: Usa la propiedad "flex-basis" para ajustar el tamaño inicial de un articulo.

La propiedad flex-basis especifica el tamaño inicial de el articulo antes de que CSS haja ajustes con flex-shrink o flex-grow.

Las unidades usadas por la propiedad flex-basis son la mismas que en otras propiedades de tamaño ( px,em,% ,etc.). El valor auto dimensiona articulos basados en el contenido.

Ajusta el tamaño inicial de las cajas usando la propiedad flex-basis. Agrega la propiedad CSS flex-basis tanto a #box-1 como a#box-2. Dale a #box-1 un valor de 10em y a #box-2 un valor de 20em.
