CSS Flexbox: Usa la propiedad "align-self"

La propiedad final para los articulos flex es align-self. Esta propiedad te permite ajustar individualmente la alineacion de cada articulo, , en lugar de ajustarlas todas en una sola vez. Esto es util desde que otras tecnicas de ajustes comunes usan las propiedades CSS float, clear, y vertical-align no funcionan en articulos flex.

align-self acepta los mismos valores como si fuera align-items y sobreescribira cualquier valor ajustado por la propiedad align-items.

Agrega la propiedad CSS align-self tanto a #box-1 como a #box-2. Dale a #box-1 un valor de center y dale a #box-2 un valor de flex-end.

Ejemplos a continuacion "align-self"

Contenedor original

Podemos ver el contenedor original y su alineacion sin la propiedad "align-self". Recordemos que si no ajustamos la direccion flex su direccion por defecnto es en row, y su alineacion por defecto sobre el main axis es al principio al igual que su alineacion sobre el cross axis es al principio, Dando como resultado que box-1 y box-2 esten juntos en la parte superior izquierda del contenedor flex.

Contenedor con "align-self" aplicado

Recordemos que align-self es una propiedad similar a align-items. Estas propiedades trabajan la alineacion de los articulos sobre el eje transversal (cross axis). Para una fila el eje transversal es vertical y para una columna es horizontal. Si no se especifica una fila o columna el contenedor flex alinea sus elementos en una fila. por tanto para este ejemplo el eje transversal es horizontal asi que el inicio del eje transversal es arriba y el final del eje transversal es al fondo del contenedor flex.

.box-container { background-color: gray; display: flex; height: 250px; } .box-1 { background-color: dodgerblue; height: 100px; width: 100px; align-self: center; } .box-2 { background-color: orangered; height: 100px; width: 100px; align-self: flex-end; }
