CSS Flexbox: Alinea elementos usando la propiedad "justify-content"

Algunas veces los articulos flex dentro de un contenedor flex no rellenan todo el espacio en el contenedor. Es comun querer decirle a CSS como alinear y espaciar los articulos flex en cierta forma. Afortunadamente. la Propiedad justify-content tiene varias opciones para hacer esto. Pero primero, hay terminologia algo importante para entender antes de revisar esas opciones.

aqui hay una imagen util mostranto una fila para ilustrat los conceptos de abajo

Recuerda que configurar un contenedor flex como una fila ubica los articulos flex lado a lado de la izquierda a la derecha. Un contenedor flex ajustado como columna coloca los articulos/items apilados en vertical de la parte superior hacia el fondo. Para cada una, la direcion de los articulos flex en la que estan acomodados es llamado main axis(eje principal). Para una fila, Esta es una linea horizontal que corta atravez de cada articulo/item. Para una columna . el eje principal es una linea vertical atravez de los articulos.

Hay varias opciones para como espaciar los articulos flex a lo largo de la linea que es el eje principal. Una de las mas comunmente usadas es justify-content: center; La cual alinea todos los articulos flex a el centro dentro del contenedor flex. otras opciones incluyen:

  • flex-start: Alinea articulos a el inicio de el contenedor flex. Para una fila, esto empuja los articulos a la izquierda de el contenedor. Para una columna, esto empuja los articulos a la cima de el contenedor.Esta es la alineacion predeterminada si no esta especificada una opcion para justify-content.
  • flex-end: Alinea articulos a el final de el contenedor flex. Para una fila, eso empuja los articulos a la derecha de el contenedor. Para una columna, esto empuja los articulos a el fondo del contenedor.
  • space-between: Alinea articulos a el centro de el eje principal, con espacio extra ubicado entre los articulos. El primero y el ultimo articulo estan muy empujados a el borde de el contenedor flex. Por ejemplo, en una fila el primer articulo esta contra el lado izquierdo de el contenedor , el ultimo articulo esta contra el lado derecho de el contenedor, luego el espacio restante es distribuido incluso alrededor de los otros articulos.
  • space-around: Similar a space-between pero el primer y ultimo articulo no estan fijos a los bordes de el contenedor, el espacio es distribuido alrededor de todos los articulos con medio espacio en cada extremo de el contenedor flex.
  • space-evenly: Distribuye espacio igual entre los articulos flex con un espacio lleno en cada extremo de el contenedor flex.

Un ejemplo ayuda a mostrar esta propiedad en accion. Agrega la propiedad cssjustify-content a el elemento #box-container(es el contenedor flex), y dale un valor de center(centro)
