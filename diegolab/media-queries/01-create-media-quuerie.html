Crear una media Querie

Media Queries son una nueva tecnica introducida en CSS3 que cambia la presentacion de el contenido basado en diferentes tama√±os de ventana. La Ventana (viewport) es el area visible del usuario en una pagina web, y es diferente dependiendo de el dispositivo usado para acceder a el sitio.

Media Queries consisten en un tipo de medio, y si este tipo de medio coincide con el tipo de dispositivo en donde el documento es visualizado, los estilos son aplicados. Tu puedes tener tantos selectores y estilos dentro de tu Media Query como tu quieras.

Aqui hay un ejemplo de una Mequia Query que retorna el contenido cuando el ancho del dispositivo es menos o igual a 100px:


@media (max-width: 100px) { /*Regla CSS*/ }


Y el siguiente Media Querie Regresa el contenido cuando la altura del dispositivo es mayor o igual que 350px:


@media (min-height: 350px) { /*Reglas CSS */ }


Recuerda, Las reglas CSS dentro de el media query es aplicado solo si el tipo de medio coincide con el dispositibo que esta siendo usado.

Agrega una media querie, de forma que la etiqueta p tenga un font-size de 10 px cuando la altura de el dispositivo es menos o igual que 800px.
