CSS Grid:Crea un espacio entre filas usando "grid-row-gap"

Puedes agregar un espacio o brecha entre las filas de una cuadricula usando grid-row-gap de la misma forma en la que agregaste un espacio entre columnas en el reto anterior.

Crea entre las filas un espacio o brecha que sea de 5px de alto.

Vista previa de los conenedores

Contenedor 1

En este ejemplo aun no hemos aplicado grid-row-gap asi que podemos observar como se comportan los articulos grid en un contenedor, sin embargo hemos resaltado en negro las propiedades que hemos usado para definir esta cuadricula en este contenedor para el este ejemplo.

.container { font-size: 40px; min-height: 300px; width: 100%; background: lightgray; display: grid; grid-template-columns: 1fr 1fr 1fr; grid-template-rows: 1fr 1fr 1fr; }

El codigo que nos permitio crear brechas entre filas y brechas o espacios entre columnas se encuentra resaltado en negro abajo y son grid-row-gap y grid-column-gap

.container2 { font-size: 40px; min-height: 300px; width: 100%; background: lightgray; display: grid; grid-template-columns: 1fr 1fr 1fr; grid-template-rows: 1fr 1fr 1fr; grid-column-gap: 5px; grid-row-gap: 5px; } .d1{background:LightSkyBlue;} .d2{background:LightSalmon;} .d3{background:PaleTurquoise;} .d4{background:LightPink;} .d5{background:PaleGreen;}
