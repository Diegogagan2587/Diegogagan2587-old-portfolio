CSS Grid: Crea tu primera cuadricula CSS

Cambia cualquier elemeto HTML en un contenedor de cuadricula si configuramos su propiedad display a grid. Esto te da la habilidad para usar todas las demas propiedades asociadas con CSS grid.

Nota:En CSS grid(cuadricula), El elememto padre se refiere a el como el contenedor y sus hijos o decendientes son llamados Articulos(items.)

Cambia el display de la etiqueta div con la clase container a grid.
