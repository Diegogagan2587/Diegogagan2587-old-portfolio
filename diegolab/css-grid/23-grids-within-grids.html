CSS Grid: Crear Cuadriculas dentro de Cuadriculas

Cambial un elemento en una Cuadricula (grid) solo afecta el comportamiento de sus decendientes directos. Asi que si cambias un decendiente directo en Grid(cuadricula), tienes una cuadricula dentro de una cuadricula.

Por ejemplo, Por ajustar las propiedades display y grid-template-column de el elemento con la clase item3, creas una cuadricula dentro de tu cuadricula.

Cambia el elemento con la clase item3 en una cuadricula con 2 columnas con ancho de auto y 1fr usando display y grid-template-column.

Ejemplo 1

Podemos ver a continuacion que nuestro ejemplo esta compuesto Por su contenedor y sus elementos. El contenedor tiene sus propiedades que determinan un contenedor de cuadricula pero ademas uno de sus elementos esta ajustado como cuadricula, podemos ver el codigo a continuacion y las propiedad de cuadricula resaltada en azul violeta:

.container { font-size: 1.5em;
min-height: 300px;
width: 100%;
background: lightgray;
display: grid;
grid-template-columns: auto 1fr;
grid-template-rows: auto 1fr auto;
grid-gap: 10px;
grid-template-areas:
"advert header"
"advert content"
"advert footer";
}
.item1 {
background: LightSkyBlue;
grid-area: header;
}
.item2 {
background: LightSalmon;
grid-area: advert;
}
.item3 { background: PaleTurquoise;
grid-area: content;
display: grid;
grid-template-columns: auto 1fr;
}
.item4 {
background: lightpink;
grid-area: footer;
}
.itemOne {
background: PaleGreen;
}
.itemTwo {
background: BlanchedAlmond;
}
