CSS Grid: ALinea todos los elementos horizontalmente usando "justify-items"

Algunas veces tu quieres que todos lo elementos en tu cuadricula CSS compartan la misma alineacion. Puedes usarlas propiedades aprendidas previamente y alinearlas individualmente, o puedes alinearlas todas de una vez horizontalmente por medio de el uso de justify-items en tu contenedor grid (contenedor de cuadricula). Esta propiedad puede aceptar todos los mismos valores acerca de los que has aprendido en los 2 desafios previos, la diferencia es o Esta en que esta movera todos los elementos en nuestra cuadricula(grid) a la alineacion deseada.

Usa esta propiedad para centrar todos nuestos elementos horizontalmente.

Ejemplo 2

Notamos como al igual que en los desafios de anteriores en este ejemplo 2 de este desafio la propiedad justify-items Alinea todos los articulos horizontalmente en el centro, nota que los fondos de los elementos ocupan el mismo espacio que el contenido de los elementos que en este caso son los dentro de ellos, a continuacion puedes ver el codigo para formar el ejemplo y la propiedad de este tema resaltada en azul violeta:

.container2 { font-size: 40px; min-height: 300px; width: 100%; background: lightgray; display: grid; grid-template-rows: 1fr 1fr 1fr; grid-template-columns: 1fr 1fr 1fr; grid-gap: 10px; justify-items: center; } .item1{background:LightSkyBlue;} .item2{background:LightSalmon;} .item3{background:PaleTurquoise;} .item4{background:LightPink;} .item5{background:PaleGreen;}
