CSS Grid: Crea diseños flexibles usando "auto-fill"

La funcion "repeat" viene con una opcion llamada "auto-fill". Esto te permite insertar automaticamente tantas filas o columnas del tamaño que deseado como sea posible deoendiendo de el tamaño de el contenedor. Tu puedes crear diseños flexibles cuando combinas "auto-fill" con "minmax", como esto:

repeat( auto-fill, minmax(60px, 1fr)) ;

Cuando el contenedor cambia de tamaño, esta configuracion se mantiene insertanto columnas de 60px y estirandolas hasta que pueda insertar otra. Nota: Si tu contenedor no puede ajustar todos tus elementos en una fila, esto las movera a bajo a una nueva fila.

En la primera cuadricula, usando auto-fill con repeat para rellenar la cuadricula con columnas que tengan un ancho minimo de 60px y un maximo de 1fr. Luego redimension la ventana para ver el auto-fill en accion.

Ejemplo 1

Nota: No mostramos las propiedades que definen los colores de los contenedores debido a que para este ejemplo son irrelevantes
A continuacion podemos ver el codigo de la figura 1 y el codigo que vamos a comparar con la figura 2 rezaltado en colo azulvioleta:

.container { font-size: 40px;
min-height: 100px;
width: 100%;
background: silver;
display: grid;
grid-template-columns: repeat(3,minmax(60px, 1fr));
grid-template-rows: 1fr 1fr 1fr;
grid-gap: 20px;
}

Nota: No mostramos las propiedades que definen los colores de los contenedores debido a que para este ejemplo son irrelevantes
Podemos observar que efectiva mente tenemos tres filas con una altura de 1fr y tenemos un numero de columnas definidas por la funcion repeat que repite las columnas con el numero de veces que autofill considere, autofill se rige por la propiedad minmax que le dice a autofill que debe rellenar el expacio con una columna siempre y cuando esta nueva columna mida como minimo 60px y como maximo 1fr
A continuacion aplicamos las propiedades vistas en esta seccion, podemos ver las propiedades de este tema resaltadas en azulvioleta:

.container2 { font-size: 40px;
min-height: 100px;
width: 100%;
background: lightgray;
display: grid;
grid-template-columns: repeat(autofill, minmax(60px, 1fr));
grid-template-rows: 1fr 1fr 1fr;
