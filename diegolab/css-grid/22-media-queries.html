CSS Grid: Usa "Media Queries" para crear diseños "responsive"

CSS Grid puede ser una foma facil de hacer tu sitio mas "responsive"(adaptable) por medio de "media queries" para reorganizar areas de grid(cuadriucla), Cambiar dimenciones de una cuadricula y reacomodar la ubicacion de los elementos.

En la vista previa, cuando el ancho de la ventana es de 300px o mas, el numero de columnas cambia de 1 a 2. El area de los anuncios ocupa entonces la columna izquierda completamente.

Cuando el ancho de ventana es de 400px o mas, haz que el adea de el header ocupe la fila de la cima completamente y el area del footer ocupe la fila de el fondo completamente.

Ejemplo 1

Para este poder lograr este ejemplo tenemos primero que recordar que esta conformado por tres piezas, Tenemos el contenedor, los elementos y la media querie Todos ellos definido en el codigo de abajo:

.container { font-size: 1.5em;
min-height: 300px;
width: 100%;
background: lightgray;
display: grid;
grid-template-columns: 1fr;
grid-template-rows: 50px auto 1fr auto;
grid-gap: 10px;
grid-template-areas:
"header"
"advert"
"content"
"footer";
}
@media (min-width: 400px) {
.container { grid-template-areas:
"header header"
"advert content"
"footer footer";
}
}
.item1 {
background: LightSkyBlue;
grid-area: header;
}
.item2 {
background: LightSalmon;
grid-area: advert;
}
.item3 { background: PaleTurquoise;
grid-area: content;
}
.item4 { background: lightpink;
grid-area: footer;
}
