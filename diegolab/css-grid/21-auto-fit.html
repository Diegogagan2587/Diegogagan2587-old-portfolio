CSS Grid: Crea diseños flexibles usando "auto-fit"

auto-fit trabaja casi identicamente a auto-fill. La unica diferencia es que cuando el tamaño del contenedor excede el tamaño de todos los elementos combinados, auto-fill se mantiene insertando filas o columnas vacias y empuja a tus elementoos a un lado, mientras que auto-fit colapsa esas filas o columnas vacias y estira tus elementos para ajustarlos a el tamaño de el contenedor.

Nota:Si tu contenedor no puede ajustar todos tus elementos en una sola fila, estas se moveran a una nueva.

En la segunda cuadricula, usa auto-fit con repeat para llenar la cuadricula con columnas que tengan un ancho minimo de 60px y un maximo de 1fr. Luego redimensiona la vista previa para ver la diferencia.

Ejemplos y Vistas previas

Ejemplo 1

A continuacion tenemos un contenedor que usaremos para comparar. Este contenedor cuenta con la propiedad auto-fill aplicada, No mostramos el codigo que muestra las propiedades de los colores de los elementos debido a que lo consideramos irrelevante para esta leccion pero si puedes ver el codigo que define la cuadricula y la propiedad de esta leccion, podras verlo en su codigo resaltado en azul violeta a continuacion:

.container { font-size: 40px;
min-height: 100px;
width: 100%;
background: lightgray;
display: grid;
grid-template-columns: repeat(auto-fill, minmax(60px, 1fr));
grid-template-rows: 1fr 1fr 1fr;
grid-gap: 10px;
}
