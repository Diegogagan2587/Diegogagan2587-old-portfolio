CSS Grid: Usa "grid-column" para controlar el espaciado.

Hasta este punto, todas las propiedades que han sido discutidas son para contenedores grid. La propiedad grid-column es la primera para usar en los articulos grid mismos.

Para controlar la cantidad de columnas que un articulo o item ocupara, tu pudes usar la propiedad grid-column en conjunto con el numero de lineas que quieras para que inicie y termine tu articulo o item.

Aqui hay un ejemplo:

grid-column: 1/3;

Esto hara que el articulo inicie en la primera linea vertical de la cuadricula en la izquierda y abarque hasta la 3ra lina de la cuadricula, ocupando 2 columnas.

Haz que el articulo con la classe item5 consuma las ultimas dos columnas de la cuadricula.

A continuacion creamos una figura con las siguientes caracteristicas:

.container { font-size: 40px; min-height: 300px; width: 100%; background: lightgray; display: grid; grid-template-columns: 1fr 1fr 1fr; grid-template-rows: 1fr 1fr 1fr; grid-gap: 10px; } .item1{background:LightSkyBlue;} .item2{background:LightSalmon;} .item3{background:PaleTurquoise;} .item4{background:LightPink;} .item5 { background: PaleGreen; }

.container { font-size: 40px; min-height: 300px; width: 100%; background: lightgray; display: grid; grid-template-columns: 1fr 1fr 1fr; grid-template-rows: 1fr 1fr 1fr; grid-gap: 10px; } .item5 { background: PaleGreen; grid-column: 2/4; }
