CSS Grid: Limita el tamaño del elemento Usando la funcion minmax

Hay otra funcion built-in(incorporada) para usar con grid-template-columns y grid-template-rows llamada minmax. Esta es usada para limitar el tamaño de elementos cuando el contenedor de cuadricula cambia de tamaño. Para hacer esto necesitas especificar el rango aceptable de tamaño para tu elemento. aqui hay un ejemplo:

grid-template-columns: 100px minmax(50px, 200px) ;

En el codigo de arriba, grid-template-columns esta ajustado para crear 2 columnas; la primera es de 100px de ancho , y la segunda tiene un ancho minimo de 50px y un ancho maximo de 200px.

Usando la funcion minmax, reemplaza 1fr en la funcion repeat con un tamaño de columna que tenga un ancho minimo de 90px y un ancho maximo de 1fr, Y redimensiona la ventana para ver el efecto.

Ejemplos y vista previa

Ejemplo 1 vista previa

Tenemos nuestro ejemplo 1 con el codigo que cambiaremos resaltado en azul violeta

.container {
font-size: 40px;
min-height: 300px;
width: 100%;
background: lightgray;
display: grid;
grid-template-columns: repeat(3, 1fr);
grid-template-rows: 1fr 1fr 1fr;
grid-gap: 10px;
}
