CSS Grid: Usa unidades CSS Grid para cambiar el tamaño de columnas y filas

Puedes usar unidades absolutas y relativas como px y em en CSS Grid para definir el tamaño de las filas y columnas. Tu puedes usar estas tan bien:

  • fr: ajusta la fila o columna a una fracion del espacio disponible.
  • auto: ajusta la fila o collumna al ancho o alto de su contenido auotmaticamente.
  • %: ajusta la fila o columna a un porcentaje del ancho de su contenedor.
Aqui esta el cogigo que genera la salida en la vista previa de el contenedor 1: grid-template-columns: auto 50px 10% 2fr 1fr;

Este retazo crea cinco columnas. La primera columna es tan ancho como su contendido, la segunda columna es de 50px, la tercera columna de el 10% de lo que mide su contenedor,y para las ultimas dos columnas; el espacio restante es dividido en tres secciones, dos estan localizadas en la cuarta columna, y una en la quinta.

Has una cuadricula con tres columnas cuyos anchos son los siguientes: 1fr, 100px, y 2fr.
