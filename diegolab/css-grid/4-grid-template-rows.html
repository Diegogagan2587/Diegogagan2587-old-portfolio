CSS Grid: Agrega filas con "grid-template-rows"

La cuadricula que creaste en el reto pasado ajustara el numero de filas automaticamente. Para ajustar las filas manual mente, usa la propiedad grid-template-rows en la misma forma que tu usaste grid-template-columns en el desafio anterior.

Agrega 2 filas a la malla que sean de 50px de alto cada una.
