CSS Grid: Alinear un Articulo horizontalmente usando "justify-self"

En CSS Grid, El contenido de cada articulo esta localizado dentro de una caja a la cual nos referimos como una celda. Tu puedes alinear la posicion de el contenido dentro de esta celda horizontal mente usando la propiedad justify-self en el articulo grid. Por defecto esta propiedad tiene un valor de stretch, La cual hara que el contenido llene todo el ancho de la celda. Esta propiedad CSS grid acepta otros valores tambien:

start: Alinea el contenido a la izquierda de la celda

center: Alinea el contenido en el centro de la celda.

end: Alinea el contenido a la derecha de la celda.

Usa la propiedad justify-self para centrar el elemento con la clase item2.

Acontuacion vemos el comportamiento una vez aplicada la propiedad justify-self, Observa que al fondo de el elemento ya no ocupa toda la celda sino mas bien el fondo coloreado se alineo en el centro ocupando unicamente el espacio ocupado por su contenido, osea alineado al centro.
nota que esta propiedad se aplico directamente al elementoy no al contenedor. Puedes ver la propiedad de la que hablamos resaltada en azul violeta a continuacion:

.container { font-size: 40px; min-height: 300px; width: 100%; background: LightGray; display: grid; grid-template-columns: 1fr 1fr 1fr; grid-template-rows: 1fr 1fr 1fr; grid gap: 10px; } .item1{background: LightSkyBlue;} .item2{background: LightSalmon;} .item2-2 { background: lightsalmon; justify-self: center; } .item3{background:PaleTurquoise;} .item4{background:LightPink;} .item5{background:PaleGreen;}
