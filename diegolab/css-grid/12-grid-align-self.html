CSS Grid: Alinear verticalmente un elemento usando "align-self"

Justo como puedes alinear un elemento horizontalmente, hay una forma para alinear un elemento verticalmente tambien. Para hacer esto, usa la propiedad align-self en un elemento. Esta propiedad acepta todas las de los mismos valores como justify-self de el ultimo desafio. Recuerda que podemos usar:
start
center
end

Alinea el elemento con la clase item3 verticalmente ajustado en end.

Obserba como al igual en con justify-self en align-self el fondo ocupa el mismo espacio que el contenido de el elemento, en este caso el espacio que o cupa el numero "3" de el elemento 3. Puedes ver a continuacion el codigo debajo, el codigo para lograr el ejemplo de abajo esta subrayado en azul violeta a continuacion:

.container { font-size: 40px; min-height: 300px; width: 100%; background: lightgray; display: grid; grid-template-columns: 1fr 1fr 1fr; grid-template-rows: 1fr 1fr 1fr; grid-gap: 10px; } .item1{background:LightSkyBlue;} .item2{background:LightSalmon;} .item3 {background: PaleTurquoise;} .item3-2 { background: paleturquoise; align-self: end; } .item4{background:LightPink;} .item5{background:PaleGreen;}
