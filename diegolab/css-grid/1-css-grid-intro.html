Introduccion a los desafios CSS Grid

Una interfaz de usuario de un sitio web ("UI") tiene dos componentes. Primero, estan los elementos visuales, Tales como el color, fuentes eh imagenes. Segundo, esta la localizacion o posicionamiento de esos elementos. En Responsive Web Design(Diseño Web adaptable), Un diseño de Interfaz de usuario debe acomodarse en varios buscadores y dispositivos diferentes accediendo al contenido.

CSS3 Introdujo Cajas Flexibles(Flexible Boxes) o Flexbox para crear diseños de paginas para una interfaz de usuario dinamica. Este es un modo de diseño que acomoda elementos en una forma predecible para diferentes tamaños de pantallas y navegadores. Aunque algo nuevo, Todos los populares y modernos navegadores soportan flexbox. Esta seccion cubre como usar flexbox y las diferentes opciones de diseño que ofrece.
