Agrega Columnas con "grid-template-columns

La simple creacion de un elemento grid no te llevara muy lejos. Necesitas definir la estructura de la cuadricula tambien. Para gregar algunas columnas a la cuadricula, usa la propiedad template-columns en un contenedor grid a como lo demostramos abajo.

.container { display: grid; grid-template-columns: 50px 50px; }

Esto le dara a tu cuadricula dos columnas que son cada una de 50px de ancho. El numero de parametros dados a la propiedad grid-template-columns indica el numero de culumnas en la cuadricula, y los valores para cada parametro indican el ancho de cada columna.

Dale al contenedor grid tres columnas que son de 100px de ancho cada una.
