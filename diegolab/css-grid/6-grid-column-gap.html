CSS Grid: Crear una brecha o espacio entre columnas usando "grid-column-gap"

Hasta aqui en las cuadriculas(grids) que tu has creado, todas las columnas han sido apretadas unas contra otras. Algunas veces tu quieres una becha o espacio(gap) entre cada columna. Para a√±adir un espacio entre las columnas, usa la propiedad grid-column-gap algo asi como esto: grid-column-gap: 10px; Esto crea 10px de espacio vacio entre todas nuestras columnas.

Dale a las columnas en la cuadricula una espacio u brecha de 20px.

Ejemplos y vistas previas del codigo

Contenedor grid 1

En este contenedor tenemos definidas las propiedades que hemos revisado previamente, aun no agregamos espacios o brechas con la propiedad grid-column-gap o grid-row-gap.

.container2 { font-size: 40px; min-height: 300px; width: 100%; background: lightgray; display: grid; grid-template-columns: 1fr 1fr 1fr; grid-template-rows: 1fr 1fr 1fr; }
