Ubicar Elementos en Areas de la cuadricula usando la propiedad "grid-area"

Despues de crear una plantilla de areas para tu contenedor de cuadricula, como mostramos el los desafios previos , puedes ubicar un elemento en tu area personalizada por medio de hacerle referencia a el nombre que le diste. Para hacer esto se usa la propiedad grid-area en un elemento como este:

.item { grid-area: header; }

Esto le permite a la cuadricula saber que quieres que la clase item1 valla a el area nombrada header. En este caso, El elemento usara la parte de arriba enteramente porque esa fila completa esta nombrada como el area header.
