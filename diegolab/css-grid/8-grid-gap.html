CSS Grid: Agrega espacios mas rapido con "grid-gap"

Descripcion de la propiedad:

grid-gap es una propiedad abreviada para grid-row-gap y grid-column-gap de los dos desafios anteriores que es mas conveniente para usar. si grip-gap tiene un valor, este creara un espacio entre todas las filas y columnas. Sin embargo, si hay dos valores, se usara el primero para ajustar el espacio entre las filas y el segundo valor para las columnas.

Usa grid-gap para introducir espacios de 10px entre las filas y espacios de 20px entre las columnas.

Vista previa y codigo:

Vista previa 1

En esta figura determinamos primero las propiedades de cuadricula de nuestro contenedor para este caso tenemos medidas de 1fraccion para cada fila y columna y en total tenemos 3 fracciones, haciendo una cuadricula de 3 partes por 3 partes dando como resultado 9 espacios. solo hemos ocupado 5 de 9. Las propiedades que definen la cuadricula estan resaltadas en negro:

.container { background: LightGray; font-size: 40px; min-height: 300px; width: 100%; display: grid; grid-template-rows: 1fr 1fr 1fr; grid-template-columns: 1fr 1fr 1fr; }
