CSS Grid: Reduce la repeticion usando la funcion repeticion.

Cuando usastes grid-template-columns y grid-template-rows para definir la estructura de una cuadricula, Introdujiste un valor por cada fila o columna que creaste.

Digamos que quieres una cuadricula con 100 filas de la misma altura. No es muy practico insertar 100 valores individual mente. Afortunadamente, hay una mejor forma por medio de usar la funcion repeat para especificar el numero de veces que quieras que tus columnas o filas sean repetidas, seguidas por una coma y el valor que deseas para repetir.

Aqui hay un ejemplo que crearia la cuadricula de 100 filas, cada fila con 50px de alto.

grid-template-rows: repeat(100, 50px) ;

Puedes incluso repetir multiples valores con la funcion repetir eh insertar la funcion entre otros valores cuando definamos una estructura de cuadricula. Aqui esta como eso luce:

grid-template-columns: repeat(2, 1fr 50px) 20px ;

Esto se traduce a:

grid-template-columns: 1fr 50px 1fr 50px 20px ;

Nota: Repetimos dos veces 1fr 50px seguido de 20px.

Usa repeat para remover la repeticion de la propiedad grid-template-columns.

Ejemplo 1

Este es nuestro contenedor origininal, hemos definido este contenedor de cuadricula a travez de grid-template-columns y grid-template-rows. Mostramos como hemos definido el valor de las filas y columnas una por una, podemos ver esto acontinuacion:

.container { font-size: 40px; min-height: 300px; width: 100%; background: lightgray; display: grid; grid-template-columns: 1fr 1fr 1fr; grid-template-rows: 1fr 1fr 1fr; grid-gap: 10px; }

Ejemplo 2

Observa que en la funcion grid-template-columns hemos cambiado la forma de definir el numero de columnas, en lugar de escribir 1fr 1fr 1fr como en el ejemplo 1, aqui simplemente hemos usado la funcion repeat para evitar escribir los valores uno por uno. Podemos observar que nos da el mismo resultado que en el ejemplo 1. puedes ver el codigo a continuacion:

.container { font-size: 40px; min-height: 300px; width: 100%; background: lightgray; display: grid; grid-template-columns: repeat(3, 1fr); grid-template-rows: 1fr 1fr 1fr; grid-gap: 10px; }
