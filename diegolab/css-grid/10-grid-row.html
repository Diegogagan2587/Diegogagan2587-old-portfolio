CSS Grid: Usa grid-row para controlar el espaciado

Por supuesto, Tu puedes hacer articulos(items) que ocupen multiples filas exactamente como puedes hacerlo con las columnas. Defines las lineas horizontales donde quieras que inicie el articulo(item) y donde termine con el uso de la propiedad grid-row sobre el articulo grid.

Haz que el elemento con la clase item5 ocupe las ultimas dos filas.

Ejemplo 2

En este ejemplo ampliamos el articulo 5 para que inicie en la fila horizontal 2 y termine en la fila horizontal 4, Esto da como resultado que nuestro artiulo o item 5 ocupe 2 espacios de ancho por dos espacios de alto, Puedes ver el codigo abajo resaltado en Azul:

.container { font-size: 40px; min-height: 300px; background: lightgray; display: grid; grid-template-columns: 1fr 1fr 1fr; grid-template-rows: 1fr 1fr 1fr; grid-gap: 10px; } .item1{background:LightSkyBlue;} .item2{background:LightSalmon;} .item3{background:PaleTurquoise;} .item4{background:LightPink;} .item5 { background: PaleGreen; grid-column: 2 / 4; grid-row: 2/4; }
