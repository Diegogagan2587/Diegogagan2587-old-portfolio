CSS Grid: Alinea todos los elementos verticalmente usando "align-items"

Usar la Propiedad align-items en un contenedor grid(contenendor de cuadricula) ajustara la alineacion verticalmente para todos los elementos en nuestra cuadricula.

Usa la propiedad align-items para mover todos los elementos a el final de cada celda.

Ejemplo 2

Podemos ver que align-items funciona igual que align-self y al igual que esta alinea nuestros elementos de forma vertical, para este caso en particualr esta alienando nuesros elementos al final de las celdas verticalmente que es lo mismo que el fondo de estas, la difenrencia entre align-items y align-self es que align-items se aplica sobre nuestros contenedor grid para ajustar todos los elementos de una sola vez mientras que align-self se aplica sobre los elementos para alinearlos uno por uno. Podemos ver el codigo que usamos para los contenedores en negro y el codigo que usamos para la alineacion resaltado en azul violeta a continuacion:

.container { font-size: 40px; min-height: 300px; width: 100%; background: lightgray; display: grid; grid-template-columns: 1fr 1fr 1fr; grid-template-rows: 1fr 1fr 1fr; grid-gap: 10px; align-items: end; } .item1{background:LightSkyBlue;} .item2{background:LightSalmon;} .item3{background:PaleTurquoise;} .item4{background:LightPink;} .item5{background:PaleGreen;}
