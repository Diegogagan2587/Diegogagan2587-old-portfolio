CSS Flexbox

Introduccion a Flexbox

Una interfaz Web de usuario (UI) tiene dos componentes Primero, Hay Elementos visuales, tal como los colores, fuentes, y imagenes. Segundo, Hay una ubicaion o posicionamiento de esos Elementos. En diseño Web Optimizado, un Diseño de interfaz de usuario(layout) debe ajustarse en varios buscadores y dispositivos acediendo al contenido.

CSS3 Introduce Cajas Felixbles (Flexible Boxes) o flexbox, para crear diseños de paginas para Una interfaz de usuario (UI) dinamica. Eso es un modo de diesaño que organiza elementos en una forma predecible para diferentes tamaños de pantalllas y navegadores. Mientras algo nuevo, Todos los buscadores/navegadores populares soportan flexbox. Esta secion cubrira como usar flexbox y las diferentes opciones de diseño que ofrece.
